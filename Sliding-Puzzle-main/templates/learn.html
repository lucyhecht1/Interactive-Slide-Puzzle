Stack and Slide Strategy

We stack the tiles in such a way now, so that all we have to do later is shuffle them around.

For rows: Place the tile with the higher number in the slot we ultimately want the lower number going. Then, put the lower-numbered tile right below it.

For columns: use the same technique, but align them side by side.
